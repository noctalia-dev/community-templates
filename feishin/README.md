# Feishin — Noctalia CSS Template

Applies the colors for your current Noctalia theme to Feishin. Requires Custom CSS enabled in Settings (Menu next to Search > Settings > Custom CSS).

This template outputs the `custom.css` file into `$XDG_CONFIG_HOME/feishin/custom.css`.

Works in light mode as well.

## Notes

- Enabling the custom Accent Color option in Feishin settings should not overwrite the CSS colors, but if it does just turn the "Use theme accent color" switch on.

- Contributions to improve the color choices are welcome as the current color picks are a bit subjective and not really aligned with normal Material guidelines. :)
